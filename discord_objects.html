Mock objects

class discord_objects.DiscordObject

Base class for mocked discord objects.

Caution

You should never instantiate a raw discord_objects.DiscordObject manually. Instead, use the create_{object} methods like accord.create_guild() in the main accord module.

id

The id of the object. Always generated automatically. Not currently in snowflake format, just a raw int

class discord_objects.Guild(name: str = None)

A mock discord object representing a discord.Guild object.

Caution

You should never instantiate a discord_objects.Guild object manually. Instead, you should use the accord.create_guild() method.

name

The name of the guild

as_dict() → dict[str, Any]

Gets the guild in dictionary format resembling the guild data sent by discord.

Returns:

A dict of the guild data

class discord_objects.Member(guild: Guild, user: User)

A mock discord object representing a discord.Member object.

Caution

You should never instantiate a discord_objects.Member object manually. The library creates members automatically when a user is used with a guild that the user is not currently a member of.

guild

The guild the user is the member of

user

The user that is the member of the guild

class discord_objects.Message(text_channel: TextChannel, author: Member)

A mock discord object representing a discord.Message object.

Caution

You should create instantiate a discord_objects.Message object. The library creates messages

accordingly as a part of it’s functionality

text_channel

The text channel the message belongs to

author

The name of the text channel

interaction

The discord.py Interaction associated with the message if any. None otherwise

class discord_objects.TextChannel(guild: Guild, name: str = None)

A mock discord object representing a discord.TextChannel object.

Caution

You should never instantiate a discord_objects.TextChannel object manually. Instead, you should use the accord.create_text_channel() method.

guild

The guild the text channel belongs to

name

The name of the text channel

class discord_objects.User(name: str = None, avatar: str = None, discriminator: str = None)

A mock discord object representing a discord.User object.

Caution

You should never instantiate a discord_objects.User object manually. Instead, you should use the accord.create_user() method.

name

The username of the user

avatar

The avatar url of the user

discriminator

The discriminator of the user. NOTE: we’ll have to see how long this lasts

as_dict() → dict[str, Any]

Gets the user in dictionary format resembling the user data sent by discord.

Returns:

A dict of the user data
